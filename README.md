# Telegram Bot "Галактика"

Бот-визитка для компании «Галактика» (производитель молочных продуктов).

## Авторы

- **Парсинг**: Степанова Анна, Блохина Валерия
- **Бот**: Мельник Наталья

## Описание

Бот предоставляет информацию о компании, брендах, продукции, новостях и контактах. Все данные загружаются из JSON-файла, полученного парсингом сайта mnogomoloka.ru на русском языке.

## Возможности

- Навигация по разделам сайта
- Информация о брендах (Свежее Завтра, Большая Кружка, Добрята, Сударыня, Тёлушка)
- Новости компании с возможностью просмотра по номерам
- Контактная информация

## Технологии

- Python 3.12
- python-telegram-bot 20.x
- BeautifulSoup4 (для парсинга)
